# Power-BI-Sales-Dashboard
📊 Global Superstore Sales Dashboard (Power BI)
Project Overview
This project presents an interactive Power BI dashboard created using the Global Superstore dataset. The dashboard provides insights into sales performance across different categories, regions, segments, and order dates.
Dataset
Dataset: Global Superstore
File Format: CSV
Dashboard Pages
1. Sum of Sales by Category
Displays the total sales generated by each product category to identify the best-performing categories.
2. Sum of Sales by Region
Shows the distribution of total sales across different regions, helping compare regional performance.
3. Sum of Sales by Segment
Visualizes total sales for each customer segment to understand customer contribution.
4. Count of Profit by Category
Displays the profit count for each product category, providing an overview of category-wise profitability.
5. Sum of Sales by Order Date
Shows the sales trend over time based on order dates, helping identify growth patterns and seasonal changes.
Tools Used
  1.Microsoft Power BI
  2.Global Superstore Dataset
Key Insights
  1.Sales vary across product categories.
  2.Regional performance differs significantly.
  3.Customer segments contribute differently to total sales.
  4.Sales trends change over time.
  5.Profit distribution can be analyzed by category.
